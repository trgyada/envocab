# VocabMaster - Ozellikler

- Excel/CSV veya manuel giris ile liste olusturma; tekrar eden kelimeleri engelleme.
- Quiz modlari: Coktan secmeli, Flashcard, Eslesme.
- Soru yonu secenekleri: Karisik, Ing -> Tr, Tr -> Ing.
- Gemini destekli ornek cumle ve ceviri; hata durumunda bildirim gosterimi.
- Bilmiyorum/Listeye ekle butonu; yanlis/ bilinmeyen cevaplar tek bir “Bilemedigim Kelimeler” havuzunda tutulur.
- Zor kelimeler filtresi (yanlis yapilan kelimelerden quiz baslatma).
- Testi Bitir ile erken cikis; dogru/yanlis sayaci ve ilerleme cizgisi.
- Animasyonlu navbar logosu; Tiklayinca ana sayfaya donus.

Not: Bu repo tek kullanici icin Firestore senkronizasyonu ve SM-2 tabanli tekrar secimi kullanir (env kurulumuna bakiniz).
